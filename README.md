# World Cup 2026 AI Match Predictor

A full-stack web app that uses Machine Learning to predict FIFA World Cup 2026 match outcomes.

## Tech Stack
- Python + FastAPI (backend)
- scikit-learn Random Forest (AI model)
- HTML + CSS + JavaScript (frontend)

## Features
- AI predictions for all 48 qualified teams
- Real 2026 FIFA rankings data
- Win/Draw/Loss probability bars
- REST API with auto-generated docs

## How to Run
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python model.py`
4. `uvicorn main:app --reload`
5. Open `frontend/index.html` in your browser
